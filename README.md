```python
def endEffectorJacobianHW3(q: list[float]) -> list[float]:
    # ‡∏î‡∏∂‡∏á‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå‡∏Å‡∏≤‡∏£‡πÅ‡∏õ‡∏•‡∏á (R, R_e) ‡πÅ‡∏•‡∏∞‡∏à‡∏∏‡∏î‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á (P, p_e) ‡∏à‡∏≤‡∏Å‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô Forward Kinematics
    R, P, R_e, p_e = FKHW3(q)

    # ‡∏™‡∏£‡πâ‡∏≤‡∏á‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå 3x3 ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö Jacobian ‡∏Ç‡∏≠‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô (J_v) ‡πÅ‡∏•‡∏∞‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏° (J_w)
    J_v = np.zeros((3, 3))  # J_v ‡∏à‡∏∞‡πÄ‡∏Å‡πá‡∏ö‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô (linear velocity)
    J_w = np.zeros((3, 3))  # J_w ‡∏à‡∏∞‡πÄ‡∏Å‡πá‡∏ö‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏° (angular velocity)

    # ‡∏™‡∏£‡πâ‡∏≤‡∏á‡πÄ‡∏ß‡∏Å‡πÄ‡∏ï‡∏≠‡∏£‡πå‡πÅ‡∏Å‡∏ô z ‡πÉ‡∏ô‡πÄ‡∏ü‡∏£‡∏°‡∏ê‡∏≤‡∏ô (Base frame) ‡∏ã‡∏∂‡πà‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏´‡∏°‡∏∏‡∏ô‡∏£‡∏≠‡∏ö‡πÅ‡∏Å‡∏ô z ‡πÉ‡∏ô‡πÄ‡∏ü‡∏£‡∏°‡∏ê‡∏≤‡∏ô‡∏Ñ‡∏∑‡∏≠ [0, 0, 1]
    z_0 = np.array([0, 0, 1])

    # ‡∏ß‡∏ô‡∏•‡∏π‡∏õ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏Ç‡πâ‡∏≠ (‡∏°‡∏µ 3 ‡∏Ç‡πâ‡∏≠) ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡∏Ñ‡πà‡∏≤ Jacobian
    for i in range(3):
        # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì‡πÅ‡∏Å‡∏ô z_i ‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i ‡πÉ‡∏ô‡πÄ‡∏ü‡∏£‡∏°‡∏ê‡∏≤‡∏ô ‡πÇ‡∏î‡∏¢‡πÉ‡∏ä‡πâ‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå‡∏Å‡∏≤‡∏£‡∏´‡∏°‡∏∏‡∏ô R ‡∏ó‡∏µ‡πà‡πÑ‡∏î‡πâ‡∏à‡∏≤‡∏Å FK
        # ‡πÇ‡∏î‡∏¢‡πÉ‡∏ä‡πâ‡∏Å‡∏≤‡∏£‡∏Ñ‡∏π‡∏ì‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå‡∏´‡∏°‡∏∏‡∏ô R ‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠ i ‡∏Å‡∏±‡∏ö‡πÅ‡∏Å‡∏ô z_0 ‡πÄ‡∏û‡∏∑‡πà‡∏≠‡∏´‡∏≤‡∏ó‡∏¥‡∏®‡∏ó‡∏≤‡∏á‡∏Ç‡∏≠‡∏á‡πÅ‡∏Å‡∏ô‡∏´‡∏°‡∏∏‡∏ô‡πÉ‡∏ô‡∏£‡∏∞‡∏ö‡∏ö‡∏û‡∏¥‡∏Å‡∏±‡∏î‡∏ê‡∏≤‡∏ô
        z_i = (R[:, :, i] @ z_0)  

        # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Jacobian ‡∏Ç‡∏≠‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô J_v ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i
        # ‡πÇ‡∏î‡∏¢‡πÉ‡∏ä‡πâ cross product ‡∏£‡∏∞‡∏´‡∏ß‡πà‡∏≤‡∏á‡πÅ‡∏Å‡∏ô‡∏´‡∏°‡∏∏‡∏ô z_i ‡πÅ‡∏•‡∏∞‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡∏≠‡∏á end-effector ‡∏•‡∏ö‡∏î‡πâ‡∏ß‡∏¢‡∏ï‡∏≥‡πÅ‡∏´‡∏ô‡πà‡∏á‡∏Ç‡∏≠‡∏á‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i
        J_v[:, i] = np.cross(z_i, p_e - P[:, i])

        # ‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Jacobian ‡∏Ç‡∏≠‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏° J_w ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏ó‡∏µ‡πà i
        # ‡πÇ‡∏î‡∏¢‡∏Ñ‡πà‡∏≤ J_w ‡∏à‡∏∞‡πÄ‡∏õ‡πá‡∏ô‡∏ó‡∏¥‡∏®‡∏ó‡∏≤‡∏á‡∏Ç‡∏≠‡∏á‡πÅ‡∏Å‡∏ô‡∏´‡∏°‡∏∏‡∏ô z_i ‡πÄ‡∏≠‡∏á (‡πÄ‡∏û‡∏£‡∏≤‡∏∞‡πÄ‡∏õ‡πá‡∏ô‡∏Ç‡πâ‡∏≠‡∏ï‡πà‡∏≠‡∏´‡∏°‡∏∏‡∏ô revolute joint)
        J_w[:, i] = z_i

    # ‡∏ô‡∏≥ Jacobian ‡∏Ç‡∏≠‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô (J_v) ‡πÅ‡∏•‡∏∞‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏° (J_w) ‡∏°‡∏≤‡∏ï‡πà‡∏≠‡∏Å‡∏±‡∏ô‡πÉ‡∏ô‡πÅ‡∏ô‡∏ß‡∏ï‡∏±‡πâ‡∏á
    # ‡πÇ‡∏î‡∏¢‡πÑ‡∏î‡πâ‡πÄ‡∏õ‡πá‡∏ô‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå 6x3 ‡∏ã‡∏∂‡πà‡∏á‡πÄ‡∏Å‡πá‡∏ö‡∏Ç‡πâ‡∏≠‡∏°‡∏π‡∏•‡∏Ç‡∏≠‡∏á‡∏ó‡∏±‡πâ‡∏á‡∏Ñ‡∏ß‡∏≤‡∏°‡πÄ‡∏£‡πá‡∏ß‡πÄ‡∏ä‡∏¥‡∏á‡πÄ‡∏™‡πâ‡∏ô‡πÅ‡∏•‡∏∞‡πÄ‡∏ä‡∏¥‡∏á‡∏°‡∏∏‡∏°
    J_e = np.vstack((J_v, J_w))

    # ‡∏™‡πà‡∏á‡∏Ñ‡∏∑‡∏ô‡∏Ñ‡πà‡∏≤ Jacobian J_e ‡∏ã‡∏∂‡πà‡∏á‡πÄ‡∏õ‡πá‡∏ô‡πÄ‡∏°‡∏ó‡∏£‡∏¥‡∏Å‡∏ã‡πå‡∏Ç‡∏ô‡∏≤‡∏î 6x3 (‡πÉ‡∏ô‡∏£‡∏π‡∏õ‡πÅ‡∏ö‡∏ö‡∏•‡∏¥‡∏™‡∏ï‡πå‡∏Ç‡∏≠‡∏á float)
    return J_e
```
